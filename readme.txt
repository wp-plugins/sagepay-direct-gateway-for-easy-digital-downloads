SagePay Direct Gateway for accepting payments on your Easy Digital Downloads Store.

== Description ==

SagePay Direct integration is essentially a white-label payment solution, giving you complete control over the way you process card payments.

This Plugin allows you to accept SagePay Payments through Sage Pay Account and when integrated with your e-commerce website, your shopper never moves from your URL.

NOTE: You need to maintain a PCI DSS certification and SSL Certificate on your site.

So this plugin helps you to accept payments with Easy Digital Downloads using SagePay Accounts.
Send us your ideas and feedback here: https://www.patsatech.com/contact-us

== Installation ==

1. Download and unzip the latest release zip file.
2. If you use the WordPress plugin uploader to install this plugin skip to step 4.
3. Upload the entire plugin directory to your `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress Administration.
